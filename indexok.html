Mammut
MAchine Micro Management UTilities

Mammut provides an object oriented abstraction of architectural features normally exposed by means of sysfs files or CPU registries. It is structured as a set of modules, each managing a specific functionality (e.g. energy profiling, CPU frequencies, topology analysis, etc...). Mammut also provides the possibility to manage remote machines by using a client server mechanism.

Main features

Energy profiling

Change clock frequency

Shutdown cores

Change processes priorities

Pin threads to cores

Force cores to idle

Topology Information
