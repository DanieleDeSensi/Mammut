bool mammut::energy::CounterCpusLinux::init ( )
virtual

Initializes the counter.

Returns
True if the counter is present, false otherwise.

I have one msr for each CPU. Since I have no guarantee that the CPU identifiers are contiguous, I find their maximum id to decide how long the array of msrs should be. For example, if a have a CPU with id 1 and a CPU with id 3, then I will allocate an array of 4 positions. In msrs[1] I will have the msr associated to CPU 1 and in msrs[3] I will have the msr associated to CPU 3.
